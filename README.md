# AI Build Crypto Dash

Стартовый каркас проекта для дашборда мониторинга пользовательских списков криптовалютных торговых пар.

## Технологический стек

- Frontend: Next.js 16, React, TypeScript, Tailwind CSS
- Backend: FastAPI (Python)
- Database: PostgreSQL 18
- Infra: Docker Compose

## Текущее состояние

- Инициализирована структура проекта по паттерну Memory Bank
- Созданы базовые контракты API и БД
- Подготовлены инфраструктурные файлы для локальной разработки

Подробности см. в `memory_bank/` и `architecture.md`.
