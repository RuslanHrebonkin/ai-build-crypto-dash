# Glossary

- **Watchlist**: персональный список торговых символов пользователя.
- **Trading Pair / Symbol**: торговая пара, например `BTC/USDT`.
- **Contracts-First**: сначала проектируются контракты API/БД, затем реализация.
- **CRUD**: Create, Read, Update, Delete.
- **DTO**: Data Transfer Object, типизированная форма данных для обмена между слоями.
