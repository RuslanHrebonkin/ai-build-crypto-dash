# Tech Context

## Взаимодействие компонентов

- **Next.js клиент** выполняет REST API вызовы к **FastAPI backend**.
- **FastAPI** обрабатывает авторизацию и CRUD-операции по символам.
- **FastAPI** взаимодействует с **PostgreSQL 18** синхронно или асинхронно:
  - через SQLAlchemy,
  - или через сырой SQL.

## Контракт API

- Основа API зафиксирована в `contracts/swagger.json`.
- Эндпоинты покрывают авторизацию и полный CRUD для пользовательских символов.

## Контракт БД

- Основа схемы БД зафиксирована в `contracts/db_schema.sql`.
- Модель данных: `users` (1) -> (N) `user_symbols`.

## Типизация фронтенда

- **TypeScript обязателен** для всех API-клиентов, DTO и UI-состояний.
- Ответы API должны описываться типами/интерфейсами для снижения ошибок интеграции.
